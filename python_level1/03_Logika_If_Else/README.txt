Sebuah program akan terasa "pintar" jika bisa mengambil keputusan. Di Python, kita menggunakan if, elif (else if), dan else.

1. Operator Perbandingan
Sebelum membuat if, kamu harus tahu cara membandingkan dua nilai:

== : Sama dengan

!= : Tidak sama dengan

>  : Lebih besar dari

<  : Lebih kecil dari

>= : Lebih besar atau sama dengan

<= : Kurang dari atau sama dengan

2. Struktur If-Else
PENTING: Python menggunakan titik dua (:) dan indentasi (spasi menjorok ke dalam) untuk menentukan baris mana yang termasuk dalam bagian if.
