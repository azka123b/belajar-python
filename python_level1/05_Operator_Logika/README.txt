Materi 5: Operator Logika (and, or, not)
Terkadang, sebuah keputusan tidak cukup hanya berdasarkan satu syarat. Di dunia Mechatronics, misalnya: "Mesin boleh menyala JIKA tombol ON ditekan DAN pintu pengaman tertutup."

Di Python, kita menggabungkan kondisi menggunakan:

and: Bernilai benar jika KEDUA syarat terpenuhi.

or: Bernilai benar jika SALAH SATU syarat terpenuhi.

not: Membalikkan keadaan (Benar jadi Salah, Salah jadi Benar).
