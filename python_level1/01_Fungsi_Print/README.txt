1. Fungsi print()
Fungsi ini digunakan untuk menampilkan sesuatu ke layar (terminal).

Teks: Harus diapit tanda kutip, misalnya: print("Halo Azka")

Angka: Langsung ditulis saja, misalnya: print(2026)

2. Variabel
Variabel adalah "wadah" untuk menyimpan data. Kamu bisa memberi nama apa saja selama tidak diawali angka dan tidak pakai spasi.

Contoh: nama = "Azka"

Di sini, nama adalah labelnya, dan "Azka" adalah isi di dalamnya.

3. Tipe Data Dasar
Ada tiga yang paling sering dipakai:

String (str): Teks, diapit kutip. Contoh: "Mechatronics"

Integer (int): Angka bulat. Contoh: 20

Float (float): Angka desimal (pakai titik, bukan koma). Contoh: 175.5
