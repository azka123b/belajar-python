Folder 11: List Functions (len, sum, max, min)
Dalam dunia teknik, kita sering berhadapan dengan sekumpulan angka (misal: data suhu sensor, tegangan baterai, atau jumlah barang). Python menyediakan fungsi bawaan untuk mengolah angka-angka di dalam list tersebut secara instan tanpa perlu menghitung manual.

1. Fungsi len() (Length)
Digunakan untuk menghitung jumlah item yang ada di dalam list.

Contoh: len([10, 20, 30]) hasilnya adalah 3.

2. Fungsi sum() (Summation)
Digunakan untuk menjumlahkan semua angka yang ada di dalam list.

Contoh: sum([10, 20, 30]) hasilnya adalah 60.

3. Fungsi max() dan min()
Digunakan untuk mencari nilai tertinggi dan terendah dari sekumpulan data.

Contoh: max([5, 12, 8]) adalah 12.

Contoh: min([5, 12, 8]) adalah 5.
