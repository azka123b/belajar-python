Cara Kerja input():
Program akan berhenti sejenak.

Menunggu user mengetik sesuatu dan menekan Enter.

Hasil ketikan tersebut akan disimpan ke dalam variabel.

PENTING: Semua hasil dari input() secara default dianggap sebagai String (Teks). Kalau kamu mau mengambil input berupa angka (untuk dihitung), kamu harus membungkusnya dengan int() atau float().
