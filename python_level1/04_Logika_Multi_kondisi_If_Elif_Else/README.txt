Materi 4: Logika Multi-Kondisi (if - elif - else)
Dalam pemrograman, seringkali pilihan kita tidak hanya dua (Benar/Salah). Terkadang ada banyak kondisi. Untuk itu, kita menggunakan elif (singkatan dari Else If).

Struktur Logika:
if: Kondisi pertama yang dicek.

elif: Dicek hanya jika kondisi sebelumnya salah. Kamu bisa punya banyak elif.

else: Dijalankan jika semua kondisi di atas salah.
